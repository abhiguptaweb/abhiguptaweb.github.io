How to copy message from alert box in IE and EDGE Browser?

To copy message from alert box in IE or EDGE browser, follow the steps below:

  • 1.) When AlertBox in focus ( alertbox opened ), press CTRL+C in your keyboard
  • (It will copy the message in clipboard.)
  • 2.) Paste (CTRL+V) to the your favourite text editor.
  • (Now, You can copy the message from Text editor.)

Windows PC Information

To find which version of Windows you are running, enter the following commands in the Command Prompt or Powershell:

  • wmic os get caption
  • wmic os get osarchitecture
